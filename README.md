# Drum Machine

A simple and interactive **Drum Machine** built using **React**.  
This project allows users to play drum sounds by clicking on drum pads or pressing keyboard keys. It also includes display output, power control, volume control, and bank switching.

## Features

- 9 clickable drum pads
- Keyboard support for keys: `Q, W, E, A, S, D, Z, X, C`
- Unique audio clip for each pad
- Display section showing the current sound name
- Power on/off control
- Volume slider control
- Bank switching for different sound sets
- Responsive layout inspired by the freeCodeCamp sample

## Technologies Used

- HTML
- CSS
- JavaScript
- React

## Project Requirements Covered

This project fulfills the main requirements of the freeCodeCamp **Drum Machine** challenge:

- `id="drum-machine"`
- `id="display"`
- 9 drum pads with class `drum-pad`
- Pads in exact order: `Q, W, E, A, S, D, Z, X, C`
- Each pad contains an HTML5 audio element with class `clip`
- Audio plays on click
- Audio plays on keyboard press
- Display updates with the associated sound name

## How to Run the Project

1. Download or clone the project
2. Open the folder in VS Code
3. Save the code in a file named `index.html`
4. Open the file in your browser

You can also use the **Live Server** extension in VS Code for a better experience.
